MeiUrbano (v1.0.0)
==========
**Linguagens:** 

* Angular 7
* @ionic/angular 4.0.0
* Ionic Native 5
* Ionic 4 Components/API Samples

---

exigências
------------

* Node 8.13+
* Npm 6+
* Ionic CLI 4.5

instalação
------------

```
$ npm install
$ ionic serve -l
```
